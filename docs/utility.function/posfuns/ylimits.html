directlabels - utility.function - Positioning Method - ylimits

Return the positions of the plot vertical limits in cm, for use as the limit argument to qp.labels.

ylimits <- function(...){
  convertY(unit(c(0,1),"npc"),"cm",valueOnly=TRUE)
}
